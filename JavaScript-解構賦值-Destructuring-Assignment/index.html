函式其餘參數 :

當其餘參數沒有傳入實際值時，會變空陣列，而不是 undefined。

其餘參數的設計是為了取代函式中的 arguments 關鍵字，arguments 實際為包含所有傳入參數的類陣列(array-like)物件，所以本身不具備陣列相關的方法，因此不建議使用 arguments。
